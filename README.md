Book Buddy is a responsive and kid-friendly web application designed for a book sharing community. It allows users to list books they've finished, discover new books from others, and manage requests for sharing or exchanging books, all within a clean and intuitive interface.


## ## ‚ú® Features

* ‚úÖ **Browse & Discover:** View all available books in a clean, grid-based layout.
* ‚úÖ **Dynamic Search & Filter:** Instantly search for books by title or author, and apply filters for genre, age group, and availability type (Free/Exchange).
* ‚úÖ **Detailed Book View:** Click on any book to see a dedicated page with more information.
* ‚úÖ **Personal Dashboard:** A comprehensive user dashboard showing:
    * **Incoming Requests:** Manage requests from other users for your books.
    * **My Shared Books:** A list of all books you have shared.
    * **My Outgoing Requests:** Track the status of books you have requested.
* ‚úÖ **Add New Books:** Users can add their own books to the platform via a centered modal form.
* ‚úÖ **Full Request Lifecycle:**
    * Users can request books from others.
    * Book owners can **Approve** or **Reject** incoming requests directly from their dashboard.
    * Approving a request updates the book's status to "Shared."
    * Rejecting a request makes the book "Available" again.
* ‚úÖ **Asynchronous Operations:** The application simulates fetching data from a server, displaying a loading state and gracefully handling potential errors.
* ‚úÖ **Responsive Design:** The layout is designed to be functional and look great on various screen sizes, from mobile phones to desktops.
